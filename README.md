## 使用方法
1. 把下载的 `chrome` 文件夹放入火狐 `配置文件夹` 中
2. 在 `about:config` 页面中搜索 `toolkit.legacyUserProfileCustomizations.stylesheets`  `svg.context-properties.content.enabled` 并设为 `true`
3. 重启火狐

## 注意事项
工具栏按钮数量仅支持默认数量（左3右4）

## Tooltip

win下全局 `文字提示` 定制需要[userChrome.js](https://github.com/MrOtherGuy/fx-autoconfig)加载

如有需要把 `tooltip` 文件夹里的内容合并到对应的目录

`profile` →  `配置目录`  `program` →  `程序目录`
